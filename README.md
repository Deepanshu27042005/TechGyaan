 TechGyaan :

TechGyaan is an Android mobile application built to help beginners and incoming college students find early guidance and direction in technology.
The idea came from a real problem I observed—many students struggle to decide where to start in tech or how to explore different domains confidently.

This project was built as a quick side project and is currently in its early development stage, with a focus on learning, iteration, and gradual improvement.

 Features :

Beginner-friendly interface

Simple and clean UI for easy navigation

Designed to provide clarity and guidance for students starting their tech journey

Lightweight and focused on core functionality

 Tech Stack :

Language: Kotlin

UI: XML

Platform: Android

Tools: Android Studio

 Purpose of the Project :

Help beginners explore different technology domains

Build confidence early in a student’s learning journey

Practice Android development fundamentals

Learn through rapid prototyping and iteration
